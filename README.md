# Predicción de niveles de obesidad con Machine Learning

Este proyecto consiste en el análisis de datos y entrenamiento de modelos de Machine Learning para predecir el nivel de obesidad de una persona a partir de hábitos alimenticios, actividad física y características físicas.

## Dataset

Se utilizó el dataset público:
https://www.kaggle.com/datasets/ankurbajaj9/obesity-levels

El conjunto de datos incluye variables como:
- Género
- Edad
- Altura
- Peso
- Consumo de agua
- Actividad física
- Hábitos alimenticios
- Tipo de transporte
- Nivel de obesidad (variable objetivo)

## Objetivo

Desarrollar un modelo capaz de clasificar el nivel de obesidad de una persona en categorías como:
- Peso insuficiente
- Peso normal
- Sobrepeso
- Obesidad tipo I, II y III

## Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn
- Google Colab / Jupyter Notebook

## Proceso del proyecto

1. Carga y exploración de datos
2. Limpieza y preparación del dataset
3. Análisis exploratorio de datos
4. Entrenamiento del modelo
5. Evaluación de resultados

## Resultados

El modelo logró clasificar los niveles de obesidad utilizando variables relacionadas con hábitos de vida y características físicas, mostrando cómo el Machine Learning puede aplicarse en problemas de salud y análisis predictivo.
